# 参考手册（3 分钟上手）

这是 `EMSimLab` 的新手入口页。  
详细内容见 `00_索引.md`，这里先给最短路径。

## 1. 先做什么

1. 启动项目（MATLAB 2025b）：
```matlab
run('run.m')
```
2. 切一个模板，改一个参数，点“运行/暂停/重置”。
3. 回来看文档主线。

## 2. 主线阅读

1. `00_索引.md`
2. `01_工程导航与运行主线.md`
3. `02_模板扩展主线.md`
4. `03_控制层调用链与回调入口.md`
5. `04_引擎与物理核扩展.md`
6. `05_UI渲染与日志排错.md`

## 3. 按任务找文档

1. 想新增模板：看 `02_模板扩展主线.md`
2. 想定位按钮逻辑：看 `03_控制层调用链与回调入口.md`
3. 想改公式：看 `04_引擎与物理核扩展.md`
4. 想改参数组件：看 `06_参数组件接口与新增组件流程.md`
5. 想排错：看 `08_FAQ_常见坑汇总.md`

## 4. 边界说明

1. 当前手册主线不把 `R2LC` 作为已接通业务功能。
2. `docs/代码原理介绍/` 是推导资料，当前按只读使用。
3. 插图先保留占位，后续可补到 `docs/参考手册/assets/`。
