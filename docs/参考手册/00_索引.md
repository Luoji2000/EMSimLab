# 参考手册索引（面向初学者）

本手册面向“会基础代码、但第一次接触本项目”的开发者，目标是让你能：

1. 跑起项目并理解主运行链路。
2. 定位核心文件，知道先读什么、后读什么。
3. 按“模板扩展”主线安全新增功能。

## 适用范围

1. 代码仓库：`EMSimLab`
2. 当前启动入口：`run.m` 与 `apps/MainApp.m`
3. MATLAB 版本：2025b（由项目维护者提供）
4. 当前不纳入实现范围：`R2LC`（先不写入业务实现）
